// Dota Heroes File
// Custom heroes for Dota 2 - Meme Mode.

// A NEW selectable hero is added (not replacing any existing hero) by:
//   - giving it a unique KV block keyed by its npc name,
//   - "override_hero" pointing at an existing hero used as a TEMPLATE
//     (model / voice / animations / base stats are inherited),
//   - a "HeroID" reusing an UNUSED gap in Valve's hero-id range (you cannot
//     mint brand-new ids; gaps below the current frontier are safest),
//   - "AttributePrimary" MUST be set: the hero-pool Lua does
//     _G[AttributePrimary] + 1 for every hero, so a missing value crashes the
//     pool build and wipes the mode-selection UI.
//   - "Enabled" "1" so it flows into the pick pool.
// Caveat (per community): if Valve ever reuses gap id 24, this would collide.
