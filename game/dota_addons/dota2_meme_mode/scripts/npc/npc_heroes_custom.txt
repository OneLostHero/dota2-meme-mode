// Dota Heroes File
// Custom heroes for Dota 2 - Meme Mode.
//
// Flasaro is a NEW, self-contained hero built on the proven custom-hero pattern:
//   - "BaseClass" points at a REAL hero (dragon_knight). This is REQUIRED so the
//     engine has an entity class to spawn and the client has something to render
//     (without it the hero is pickable but never spawns and the portrait is black).
//     BaseClass INHERITS; it does not override -- Dragon Knight stays separate.
//   - "HeroID" is a high, unused id (250). The recycled gap 24 did not spawn.
//   - No "Facets" block (facets were removed from the game; working customs omit it).
//   - "override_hero" is NOT used (it modifies the target hero in place).
//   - Model/voice/stats come from Dragon Knight so Flasaro looks like DK.
